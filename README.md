# AI-Tech Support
>A whatsapp tech support chat bot system with ai models to classify requests and provide responses.
## Features
- Whatsapp chatbot
- [distilbert-base-uncased](https://huggingface.co/distilbert/distilbert-base-uncased) classifier
- [google/flan-t5-small](https://huggingface.co/google/flan-t5-small) for text generation
- admin panel
